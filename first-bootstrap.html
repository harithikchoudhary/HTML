Types of Container Tags

Go to bottom

Container Tags

Container Tags - Which don't have their own content. Examples like:
  • table
  • body
  • div

Standalone Tags

Standalone Tags - Which need not be closed. Examples like:
  • br
  • img
  • hr

Content Tags

Content Tags - Which must be opened and closed. The content is entered in between opening and closing tag. Examples like:
  • h1
  • td
  • p
  • b
  • i

Parent and Child Tags

Parent and child tags refer to the hierarchical relationship between tags, where a parent tag contains one or more child tags. Examples like:
  • table
  • tr
  • td

Tags with Attributes

Tags have attributes. Examples are:
  • id
  • class
  • style

Heading Tags

Heading tags are used to define headings of different levels in a document. The <h1> tag defines the most important heading, and <h6> defines the least important heading.

Text Formatting

Text Formatting:
  • <b> Bold </b>
  • <i> Italic </i>
  • <u> Underline </u>
  • <strike> Strike </strike>
  • x<sub>2</sub> x2
  • y<sup>2</sup> y2

Punctuation Tags

  • You are a Trillionare Anil Sir
  • <q>You are a Trillionare </q>
  • <cite>Anil Sir </cite>

Punctuation tags are used to add quotations and citations in the text, making it easier to format quoted text and references properly.
